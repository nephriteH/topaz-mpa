# topaz-mpa

# 多页面项目

## 技术栈 vite+vue3.ts+prettier+eslint+node.js

## [多页面项目搭建文档](https://www.jianshu.com/p/ad28486eb8e3)

## 项目安装

```sh
npm install
```

### 本地调试

```sh
# 启动所有页面
npm run dev-all
# 启动单独页面
npm run dev --page=a
```

### 构建项目

```sh
# 构建所有页面
npm run build-all
# 构建单独页面
npm run build --page=a
```
